# AI 图片文案生成器

基于 Flask 和 AI 技术的图片文案生成工具，支持多种风格的文字描述生成。

## 功能特点

### 多样化的生成风格
- 🎭 **心情描述**：分析图片情感，生成贴切的心情描述
- 😏 **讽刺风格**：以幽默讽刺的视角解读图片
- 📝 **诗意描述**：将图片转化为富有诗意的文字
- 📖 **故事创作**：基于图片编织有趣的故事

### 技术特性
- 支持多种图片格式：JPG、PNG、GIF、WEBP
- 图片大小限制：5MB
- 自动图片压缩和格式验证
- API 接口安全验证
